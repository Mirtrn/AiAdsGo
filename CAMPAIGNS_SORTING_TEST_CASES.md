# Campaigns 页面排序功能测试用例

## 修复说明

**修复时间**: 2025-12-29
**问题**: 排序功能被绕过
**根本原因**: `fetchCampaigns()` 直接设置 `filteredCampaigns`，绕过 useEffect 的排序处理
**修复方式**: 移除 `fetchCampaigns()` 中的 `setFilteredCampaigns()` 调用
